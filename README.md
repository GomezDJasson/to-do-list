# 📝 To Do List App

Aplicación de gestión de tareas desarrollada con **React**, diseñada para ser rápida, intuitiva y adaptable tanto a escritorio como a dispositivos móviles.

---

## 🚀 Demo

🔗 [Ver aplicación en producción](https://to-do-list-one-dun-23.vercel.app/)

---

## 📸 Preview

![App Screenshot](/src/assets/todo-dark.png, /src/assets/todo-light.png)

---

## ✨ Funcionalidades

* ✅ Crear, editar y eliminar tareas
* ✅ Marcar tareas como completadas
* ✅ Filtros: todas, pendientes y completadas
* ✅ Múltiples listas (trabajo, casa, estudio, etc.)
* ✅ Editar y eliminar listas
* ✅ Drag & Drop para ordenar tareas
* ✅ Contador de tareas pendientes por lista
* ✅ Barra de progreso de tareas completadas
* ✅ Modo oscuro (Dark Mode)
* ✅ Diseño responsive (desktop + móvil)
* ✅ Input fijo en móvil para mejor UX

---

## 🛠️ Tecnologías

* **React** (Hooks)
* **Vite**
* **Tailwind CSS**
* **Framer Motion**
* **@dnd-kit** (Drag & Drop)
* **Lucide React** (iconos)

---

## 📱 Responsive Design

La aplicación está optimizada para:

* 💻 Escritorio (sidebar de listas)
* 📱 Móvil (selector de listas + input fijo)

---

## ⚙️ Instalación

```bash
# Clonar repositorio
git clone https://github.com/GomezDJasson/to-do-list.git

# Entrar al proyecto
cd to-do-list/

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
```
